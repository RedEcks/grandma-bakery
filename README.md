# Grandma's Bakery

Welcome to "Grandma's Bakery," your delightful online destination for scrumptious baked goods that taste just like home! Our e-commerce store is meticulously designed to bring you the irresistable flavors of grandma's kitchen with the convenience of online hopping. Whether you're seeking the perfect pastries, cookies, or cakes, "Grandma's Bakery" is your virtual haven for sweet, homemade goodness delivered right to your doorstep.

# Features

- Uses SQL database to store information. 
- Uses client-side routing for seamless trasitions to different web pages.
- Uses PostgreSQL, Express.js, React, Node.js.
- Renders different baked goods in a database to the DOM.
- User can add selected good to shopping cart.
- User can purchase items in their account.

# Requirements 

- Uses React.

# How to use

- Fork and clone.
- Use "npm start."
